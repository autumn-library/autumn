# АннотацияУсловиеНаСвойство

## ПриСозданииОбъекта

```bsl
// Аннотация условной регистрации желудя на основе значения свойства конфигурации.
//
// Аналог `@ConditionalOnProperty` из Spring Boot. Если условие не выполнено,
// желудь не будет зарегистрирован в контексте Поделки.
//
// Размещается над конструктором класса или над методом `&Завязи`.
//
// Параметры:
//   Значение - Строка - Ключ свойства в конфигурации (например, "МоиНастройки.Включено").
//   ОжидаемоеЗначение - Строка - Ожидаемое значение свойства. По умолчанию "true".
//   СопоставлятьЕслиОтсутствует - Булево - Если Истина, условие считается выполненным
//                                           при отсутствии свойства. По умолчанию Ложь.
//
// Пример:
//
//  1. Регистрация, если свойство "МояФича.Включена" равно "true":
//  &УсловиеНаСвойство("МояФича.Включена")
//  &Желудь
//  Процедура ПриСозданииОбъекта()
//  КонецПроцедуры
//
//  2. Регистрация, если свойство "Режим" равно "Продакшн":
//  &УсловиеНаСвойство(Значение = "Режим", ОжидаемоеЗначение = "Продакшн")
//  &Желудь
//  Процедура ПриСозданииОбъекта()
//  КонецПроцедуры
//
//  3. Регистрация, даже если свойство отсутствует:
//  &УсловиеНаСвойство(Значение = "Кеш.Включен", СопоставлятьЕслиОтсутствует = Истина)
//  &Желудь
//  Процедура ПриСозданииОбъекта()
//  КонецПроцедуры
//
Процедура ПриСозданииОбъекта(Значение = "", ОжидаемоеЗначение = "true", СопоставлятьЕслиОтсутствует = Ложь) 
```

## Совпадает

```bsl
// Проверить, выполнено ли условие на свойство конфигурации.
//
// Параметры:
//   Поделка - Поделка - Контекст приложения.
//
//  Возвращаемое значение:
//   Булево - Истина, если условие выполнено, иначе Ложь.
//
Функция Совпадает(Поделка) 
```

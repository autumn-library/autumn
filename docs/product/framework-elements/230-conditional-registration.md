# Условная регистрация компонентов

ОСень поддерживает условную регистрацию желудей на основе значений свойств конфигурации. Это позволяет гибко управлять составом приложения в зависимости от настроек, не меняя код.

## Условие на свойство

Аннотация `&УсловиеНаСвойство` — аналог `@ConditionalOnProperty` из Spring Boot. Позволяет зарегистрировать желудь только при определённом значении свойства в `autumn-properties.json`.

::: code-group

```bsl [Классы/МойЖелудь.os]
&УсловиеНаСвойство("МояФича.Включена")
&Желудь
Процедура ПриСозданииОбъекта()

КонецПроцедуры
```

```json [autumn-properties.json]
{
    "МояФича": {
        "Включена": "true"
    }
}
```

:::

Если значение свойства `МояФича.Включена` равно `"true"`, желудь будет зарегистрирован. Иначе — нет.

### Кастомное ожидаемое значение

По умолчанию ожидается значение `"true"`. Это можно изменить:

```bsl
&УсловиеНаСвойство(Значение = "Режим", ОжидаемоеЗначение = "Продакшн")
&Желудь
Процедура ПриСозданииОбъекта()

КонецПроцедуры
```

### Поведение при отсутствии свойства

Если свойство отсутствует в конфигурации, по умолчанию желудь **не** регистрируется. Это поведение можно изменить:

```bsl
&УсловиеНаСвойство(Значение = "Кеш.Включен", СопоставлятьЕслиОтсутствует = Истина)
&Желудь
Процедура ПриСозданииОбъекта()

КонецПроцедуры
```

В этом случае желудь будет зарегистрирован, даже если свойство `Кеш.Включен` не задано.

### Условие на Завязи

Условие можно повесить на метод `&Завязи` в `&Дубе`:

```bsl
&Дуб
Процедура ПриСозданииОбъекта()

КонецПроцедуры

&УсловиеНаСвойство("Кеш.Включен")
&Завязь
Функция КешированныйСервис() Экспорт
    Возврат Новый КешированныйСервис();
КонецФункции
```

## Несколько условий

На один желудь можно повесить несколько условий. Желудь будет зарегистрирован, только если **все** условия выполнены:

```bsl
&УсловиеНаСвойство("Фича1.Включена")
&УсловиеНаСвойство("Фича2.Включена")
&Желудь
Процедура ПриСозданииОбъекта()

КонецПроцедуры
```

## Создание собственных условий

Механизм условий расширяемый. Вы можете создать собственную аннотацию-условие:

1. Создайте класс аннотации, наследующий `&Условие`:

```bsl
&Условие("МоеУсловие")
&Аннотация("МоеУсловие")
Процедура ПриСозданииОбъекта(МойПараметр)
    _МойПараметр = МойПараметр;
КонецПроцедуры
```

2. Реализуйте метод `Совпадает(Поделка)`:

```bsl
Функция Совпадает(Поделка) Экспорт
    // Ваша логика проверки условия
    Возврат <результат проверки>;
КонецФункции
```

3. Используйте аннотацию на желудях:

```bsl
&МоеУсловие("Параметр")
&Желудь
Процедура ПриСозданииОбъекта()

КонецПроцедуры
```
